# flask Server
## python3
### Downloads
* sudo apt install python3
* sudo apt install python3-pip
* sudo apt install python3-flask
## flask
### Install
* pip3 install flask
* flask --version (check)
### Run(Default)
* cd default
* flask run
  * exe file name must be 'app.py'
### Run(Server)
* python3 app.py
